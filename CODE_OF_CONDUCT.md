# Code of Conduct

## Our Commitment

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent, ask clarifying questions, and discuss ideas on technical merit.
- Give actionable, concrete feedback.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Publishing private information without permission.
- Trolling, intimidation, or abusive language.

## Enforcement

Project maintainers may remove comments, reject contributions, or ban participants for behavior that violates this code.

## Scope

This code applies in all project spaces, including issues, PRs, discussions, and other public/private project communication channels.
